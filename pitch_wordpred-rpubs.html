Introduction to WordPred

WordPred is a web app built as part of the Capstone Project of the Data Science Especialization at Coursera.

The app was created using R language and the shiny package with the goal of predicting the next word in a phrase submitted by the user

How WordPred works

After the user inputs a phrase and press the âPredictâ buttom, three words are returned as possible next words.

To return these predictions the phrase submitted by the user is parsed and searched in WordPred's database.

If a match is found, the three most frequent last words are returned. If no match is found, three of the most frequent words are returned

How WordPred was created

The text data used to create WordPred was extracted from news, twitter and blogs sources. First, the data was transformed into corpus, tokenized and ngrams of 2 to 5 words were generated.

Next, a dataframe was created using the ngrams, its last words and their frequencies, alongside with a function. Given an input, the function uses a Back-Off strategy, therefore, it extracts the last words up to 4, searches the data frame on the 'input' column and returns the words from the 'prediction' column, based on the 'frequency' column. If no match is found, this process is repeated with a smaller ngram each time.
